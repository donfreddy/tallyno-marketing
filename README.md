# Tallyno — Site Marketing

[![Nuxt UI](https://img.shields.io/badge/Made%20with-Nuxt%20UI-00DC82?logo=nuxt&labelColor=020420)](https://ui.nuxt.com)

Site vitrine de [Tallyno](https://tallyno.app), l'app qui simplifie les comptes entre amis.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

Ouvre `http://localhost:3000`.

## Build

```bash
pnpm build
```

Prévisualisation :

```bash
pnpm preview
```

## Tech Stack

- **Framework :** Nuxt 4
- **UI :** Nuxt UI 4 (Tailwind CSS 4)
- **Polices :** DM Sans, DM Mono
- **Déploiement :** Statique (prerender)
